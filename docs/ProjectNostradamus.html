Class: ProjectNostradamus

ProjectNostradamus

Class representing whole game. Once created it displays game in defined container.

Constructor

new ProjectNostradamus(width, height, renderer, parent)

Create a game.
Parameters:
Name Type Description
width number | string The width of game in pixels. If given as a string the value must be between - and 100 and will be used as percentage width.
height number | string The height of game in pixels. If given as a string the value must be between - and 100 and will be used as percentage height.
renderer number Which renderer to use: Phaser.AUTO will auto-detect, Phaser.WEBGL, Phaser.CANVAS or Phaser.HEADLESS (no rendering at all).
parent string | HTMLElement The DOM element into which this games canvas will be injected. Either a DOM ID (string) or the element itself.
