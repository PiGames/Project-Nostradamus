Class: Entity

Entity

Class representing any entity in game world. It derives after Phaser.Sprite.

Constructor

new Entity(game, x, x, imageKey, frame)

Create a Entity. Set its anchor to center, enable arcade physics on it add entity to existing game world.
Parameters:
Name Type Description
game object A reference to the currently running game.
x number The x coordinate (in world space) to position the Sprite at.
x number The y coordinate (in world space) to position the Sprite at.
imageKey string This is the key to image used by the Sprite during rendering.
frame number If this Sprite is using part of a sprite sheet or texture atlas you can specify the exact frame to use by giving a numeric index.
